This is part of [Public Audio](https://excavations.digital/projects/public-audio/)'s first phase, a research project by [MatGuzzo](http://lattes.cnpq.br/6679644870926297) in his masters in UCSD.

It basically displays content crawled since 2019 from the most viewed left-winged digital newspapers in Brazil.
It also allows to favorite highlights when logged in, but visualization of favs is currently disabled.
